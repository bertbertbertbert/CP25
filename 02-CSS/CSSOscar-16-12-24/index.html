Jerarquia en Css. - Restricciones

Los navegadores aplican estilos a los documentos Html por jerarquía basada en:

  1. La Cascada
  2. La especificidad
  3. La Herencia

La Cascada es el procedimiento con el que un navegador aplica los estilos y el orden de importancia o restricción es el siguiente:

  1. Origen del código
    • Reglas del User Agent
    • Reglas del usuario
    • Reglas desarrollador
  2. La especificidad del selector
  3. Orden de aparición

La especificidad es la fuerza restrictiva o peso de un selector.

  • Estilo en línea 1
  • Identificadores 0,1
  • Clases, atributos, pseudoclases 0,0,1
  • Etiquetas html 0,0,0,1
  • !important- Salta toda la jerarquía.

La herencia es la capacidad que tienen los elementos padre de hacer heredar a sus hijos, sus propiedades. Valores inherit o heredados.

Resetear navegadores

Ante la diferente interpretación del aspecto de los elementos de html que utiliza cada navegador, nace la necesidad de borrar o resetear todas estas propiedades nativas de los navegadores para dejar el control a los desarrolladores.
Existen en el mercado diversos archivos css que funcionan como reseteadores estandard, cómo normalize.css o reset.css.
