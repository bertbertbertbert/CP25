Jerarquía en CSS -restricciones-

Los navegadores aplican estilos a los dcumentos HTMLS por jerarquia basada en:

  1. Cascada
  2. Especifidad
  3. Herencia

La cascada es el procedimiento con el que un navegador aplica estilos y el orden de importancia o restriccion es el siguiente:

  1. Origen del codigo
    • Reglas del User Agent
    • Reglas del usuario
    • Reglas desarrollador
  2. Ka especificidad del selector
  3. Oden de aparición

La especidicidad es la fuerza restrictiva o peso de un selector

  • Estilo en línea 1
  • id 0,1
  • clases, atributos, pseudoclases 0,01
  • Etiquetas 0,0,0,1
  • !important - Es el amo. Se salta toda la jerarquía

La Herencia es la capicada que tienen los padres de hacer heredar a sus hijos sus propiedades. "Valores inherit o hererados".
